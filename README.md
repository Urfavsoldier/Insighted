# InsightED

Многоролевой AI-портал для Aqbobek Lyceum с аналитикой успеваемости, Smart Schedule и Kiosk Mode.

## Возможности
- AI-анализ и предиктивные рекомендации
- Mock API BilimClass
- Smart Schedule
- Kiosk Mode
- Тёмная / светлая тема
- Роли: ученик, учитель, родитель, администрация

## Быстрый запуск
```bash
npm install
npm run dev
