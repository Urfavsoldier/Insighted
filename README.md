# InsightED

## Запуск проекта

### Варианты установки
```bash
Вариант 1 — через GitHub
git clone https://github.com/Urfavsoldier/Insighted.git
cd Insighted
npm install
npm run dev
Вариант 2 — через ZIP

Скачайте архив через GitHub → Code → Download ZIP, распакуйте его, откройте папку проекта и выполните:

npm install
npm run dev
Открыть в браузере
http://localhost:3000
